# dstack
> Development stack for AI-assisted multi-repo work.

**Persistent memory. Quality gates. Deployment automation.**

## Install
```bash
cargo install dstack-cli
```

## Commands
- **`dstack memory save/load/query/export`** — Persistent context
- **`dstack sync --status`** — Git status across repos
- **`dstack deploy <service>`** — Build → restart → smoke test
- **`dstack audit --pre-commit`** — 5-question quality gate
